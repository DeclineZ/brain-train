# Floating Pool Balls - Ball Images

This directory contains the colored ball images for the Floating Pool Balls game.

## Required Images

The game expects 4 PNG images named as follows:

1. **ball-1.png** - Coral colored ball (pinkish-red)
   - Hex: #FF6B6B
   - Gradient: Light coral to dark coral

2. **ball-2.png** - Mint colored ball (teal/turquoise)
   - Hex: #4ECDC4
   - Gradient: Light mint to dark mint

3. **ball-3.png** - Yellow colored ball
   - Hex: #FFE66D
   - Gradient: Light yellow to golden yellow

4. **ball-4.png** - Lavender colored ball (purple)
   - Hex: #6C5CE7
   - Gradient: Light lavender to dark lavender

## Image Specifications

- Size: 200x200 pixels (recommended)
- Format: PNG with transparency
- Style: 3D sphere appearance with:
  - Radial gradient for 3D effect
  - Subtle shadow for depth
  - White number area in center (for text overlay)
  - Clean, vibrant colors suitable for light theme

## Temporary Solution

If you don't have custom images, you can:
1. Use the SVG files in this directory as references
2. Generate images using online tools like:
   - https://www.flaticon.com (search for "pool ball")
   - Create simple circles with radial gradients in any image editor
3. Use colored circle images temporarily

## How to Convert SVG to PNG

If you want to use the SVG files:
1. Open SVG in browser or image editor
2. Export/screenshot as PNG
3. Save as ball-1.png, ball-2.png, ball-3.png, ball-4.png
4. Place in this directory
